模块2 拼图功能
<img width="2559" height="1528" alt="image" src="https://github.com/user-attachments/assets/aebe6539-9feb-4f80-a0f9-cb1f286e1432" />
支持多种可调节模板，双击边缘线即可进入角度调节，相比于只能横平竖直的拼图更加灵活，也可以通过拖拽调整展示比例。支持大尺寸图片输出。
对于大量的图片目前为了优化考虑仅支持固定的网格堆积。
